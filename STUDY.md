## 1.1 Creating a Github Repository

- git remote origin (git 저장소 주소)

  현재 작업 영역을 github repository에 연결

- git add .

  현재 작업 영역에 생성된 모든 파일을 선택

- git commit -m "커밋 메시지"

  작업 내용을 정리하여 커밋

- git push origin master

  add된 파일들을 저장소로 push
